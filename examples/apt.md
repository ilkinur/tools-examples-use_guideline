# APT

### apt-cache
`apt-cache search toolname` - Paket adında və ya təsvirində axtarış edir.  
`apt-cache show toolname` - Paket haqqında detallı məlumat verir.  
`apt-cache policy toolname` - Paketin versiya vəziyyətini və repo prioritetini göstərir.  
`apt-cache depends toolname` - Paketin asılılıqlarını göstərir.
